Acerca de este proyecto

Este proyecto fue desarrollado como parte del programa de Ingeniería de Computación en Syspro. Su objetivo es integrar habilidades de diseño web, programación y bases de datos para crear una solución funcional y accesible.

Exploramos tecnologías modernas y buenas prácticas para brindar una experiencia óptima al usuario, demostrando el aprendizaje y la innovación en el desarrollo de soluciones digitales.
